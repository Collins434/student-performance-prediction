
# 🎓 Student Performance Prediction App

This project predicts whether a student will **Pass or Fail** using a **Logistic Regression Machine Learning model**.

## Features
- Study Hours
- Attendance
- Previous Score
- Assignments Completed
- Participation
- Internet Access
- Study Environment

## Run Locally

Install dependencies:

pip install -r requirements.txt

Train model:

python train_model.py

Run Streamlit:

streamlit run app.py

## Deployment

You can deploy using:

- Streamlit Cloud
- Render
- HuggingFace Spaces
- GitHub

Upload the repository and select **app.py** as the entry point.
